How to read it. Each fight in the measurable slice gives both sides a random artifact of this tier (never mirrored, never the same creature roster), so a win% is that artifact relative to the field, not versus "no artifact" — the field centers a touch under 50% from draw handling and roster noise. Read the ranking and lift, and treat the whisker (95% Wilson CI) as the real signal: green when it clears 50%, red when it sits below, grey when it straddles. Per-cohort and per-map slices carry fewer fights, so their whiskers widen. Maps: Open = Normal board, Lava = lava center, Mountains = two destructible blocks; Live aggregates all three.
